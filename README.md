
# Shop Billing System

## Description

A **C++ console-based application** for managing product sales and generating bills.
This system allows you to add products, calculate totals, and produce an itemized bill easily.

## Features

- Add items with name, quantity, and price
- Calculate total price
- Generate itemized bill

## Technologies
- C++

## How to Run

1. **Install a C++ Compiler**
   Make sure you have a C++ compiler installed:

   * **Windows:** [MinGW](https://www.mingw-w64.org/) or Visual Studio
   * **Linux:** `g++` (install via `sudo apt install g++`)
   * **Mac:** Xcode Command Line Tools (`xcode-select --install`)

2. **Download the Project**
   Clone or download the repository:

   git clone https://github.com/lensen-degife/shop-billing.git
   cd shop-billing

3. **Compile the Code**
   In the terminal, run:
   
   g++ main.cpp -o shop-billing
   

5. **Run the Program**

  **Windows:**

     shop-billing.exe

  **Linux/Mac:**

     ./shop-billing

6. **Use the Application**
   Follow the on-screen prompts to:

   - Add items
   - View total price
   - Generate an itemized bill
